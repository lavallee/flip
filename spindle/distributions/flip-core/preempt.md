# flip — reporter's notebooks

This surface has the flip notebook skills bound: `notebook-create`,
`notebook-source`, `notebook-log`, `notebook-audit`, `notebook-handoff`,
`notebook-lessons`. When the work involves research that should carry
lineage — sources worth keeping, claims worth verifying, decisions worth
recording — reach for them. The doctrine in one line: capture before cite,
judge before verify, record the why, preserve keys you don't own, never
edit generated views.
